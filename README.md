# jsSimpleApp

Набор простейших одностраничных приложений

<b>1. ipApp</b>
Позволяет получить IP-адрес пользователя и минимальные сведения, доступные по IP.

<b>2. newsApp</b>
Приложение генерирует новости по категориям.

<b>3. todoApp</b>
Позволяет добавлять задачи в список дел, отмечать их как выполненные и удалять.
